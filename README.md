# FinCalendar — Бюджет Трекер

Мобильное приложение для учёта финансов. Доступно для Android (APK) и iPhone (PWA).

## 📱 Скачать для Android

Скачайте последнюю версию APK из раздела [Releases](https://github.com/levkinabas-star/budjet-treker/releases).

### Установка APK
1. Скачайте файл `fincalendar.apk`
2. Откройте файл в загрузках
3. Разрешите установку из неизвестных источников: **Настройки → Безопасность → Неизвестные источники**
4. Нажмите «Установить» и дождитесь завершения

> ⚠️ Для установки APK-файлов необходимо разрешить установку из неизвестных источников в настройках Android.

---

## 🍎 Установка на iPhone / iPad (PWA)

На iOS приложение устанавливается как Progressive Web App через Safari.

1. **Откройте Safari** — перейдите на сайт приложения. Другие браузеры не поддерживают установку PWA.
2. **Нажмите «Поделиться»** — найдите значок Share (квадрат со стрелкой) в нижней панели.
3. **Выберите «На экран "Домой"»** — прокрутите меню вниз и нажмите "Add to Home Screen".
4. **Подтвердите** — нажмите «Добавить». Иконка появится на рабочем столе.

### Что вы получите:
- ✅ Иконка на рабочем столе
- ✅ Полноэкранный режим без адресной строки
- ✅ Работа без интернета
- ✅ Быстрый запуск как у нативного приложения
